# HTML Structure Practice Project

## هدف پروژه
این پروژه برای تمرین ساختار HTML5 و یادگیری semantic tags ساخته شده است.

---

## نکات مهمی که یاد گرفتم:

### 1. ساختار استاندارد HTML
- استفاده از `DOCTYPE`
- استفاده از `head` و `body`
- تنظیم `meta charset` و `viewport`

---

### 2. Semantic Tags
- header برای معرفی
- nav برای منو
- main برای محتوای اصلی
- article برای محتوای مستقل
- section برای بخش‌بندی محتوا
- aside برای محتوای جانبی
- footer برای انتهای صفحه

---

### 3. نکات مهم مصاحبه‌ای
- فقط یک `main` در هر صفحه باید باشد
- استفاده از semantic به جای div اهمیت دارد
- alt در تصاویر برای accessibility ضروری است
- HTML فقط ساختار است، نه طراحی (CSS جداست)

---

این پروژه پایه‌ای برای ساخت صفحات حرفه‌ای وب است.
